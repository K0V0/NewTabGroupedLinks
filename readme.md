NewTabGroupedLinks
==================

*Google Chrome & Chrome based browsers extension to use new tab as start page with links.*
<br>
*Links are sorted into colums/blocks by categories (groups) in which they can occur independently
or in sub-categories that can be expanded or folded by default by user preference.*
<br>
*User should be able to define multiple workspaces (environments) with links that he can switch into.*

Build & Run
-----------

- install ```node``` version 20+
- install Vite (for builds) and TypeScript
  ```
  npm init -y
  npm install -D vite typescript
  ```
- ```npm run build``` will generate unpacked extension under ```/dist```
